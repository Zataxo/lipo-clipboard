# Security Policy

## Supported Versions
#v1.1.2

We actively support security updates for the latest releases of Lipo Clipboard.

## Reporting a Vulnerability
If you discover any security-related issues or vulnerabilities, please do not open a public GitHub issue. Instead, please report it privately by contacting the maintainer via email at: [hassan.og@outlook.com]

We take security seriously and will investigate and respond to all valid reports as quickly as possible.
